#  Birthday Celebration Web Application

An interactive and visually engaging birthday celebration web application built using **Flask**, designed to deliver a personalized and delightful user experience through dynamic rendering and rich animations.

---

##  Overview

This project allows users to enter a name and instantly generate a customized birthday celebration interface featuring animated visuals, interactive elements, and modern UI design.

It demonstrates the integration of backend logic with frontend creativity to build a complete, user-focused web application.

---

##  Key Features

- **Personalized Greetings** – Dynamic name-based birthday message rendering
-  **Confetti Effects** – Real-time celebration using Canvas Confetti
-  **Animated Balloons** – Floating balloons generated dynamically via JavaScript
-  **Interactive Character** – Animated teddy bear with toggle dance effect
-  **Modern UI Design** – Glassmorphism card with animated gradient backgrounds
-  **Smooth User Experience** – Clean transitions and responsive interactions

---

##  Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, JavaScript
- **Libraries:** Canvas Confetti (CDN)
- **Styling:** Custom CSS animations & glassmorphism design
